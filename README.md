# ManagerTasks
Gerenciador de projetos e tarefas

##Dependências e pré-requisitos

* [Node.js](https://nodejs.org/en)
* [MongoDB](https://www.mongodb.org)
* [Bower](http://bower.io)

## Instalação

Abra o terminal e navegue até a pasta onde foi realizado o clone do repositório.

Devem ser executados os seguintes passos no terminal:

`npm install`

`bower install`

## Execução

Para executar e testar a aplicação, pode ser realizado da seguinte forma:

`npm start`, `node start.js` ou `node server.js`
